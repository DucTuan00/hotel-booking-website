# Hotel Booking System
## Features

**Guest / User**
- Browse & search rooms with real-time availability calendar
- Book rooms, restaurant tables, and spa sessions in one flow
- Online payment via **VNPay** and **MoMo** (sandbox)
- Google OAuth 2.0 sign-in
- AI-powered trip planner (Google Gemini)
- Send automation email with Resend
- Write reviews, manage favorites
- View & cancel bookings

**Admin Dashboard**
- Manage rooms, amenities, images (Cloudinary)
- Handle bookings and reservation calendar
- Manage restaurant & spa services
- Role-based access control

**Mobile (Android)**
- Same React codebase wrapped with **Capacitor**
- Works on emulator and physical devices (WiFi / USB ADB)
